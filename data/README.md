# Data Sources

## Air Data Solutions (ADS)

Air Data Solutions is an aerial imaging company operated by Stephen Ambagis in Tucson, AZ. The company provides high-resolution aerial imagery captured via aircraft.

### Available Data

#### Tumamoc Hill Orthorectified Mosaic (February 2023)

Orthorectified aerial mosaic of Tumamoc Hill captured in February 2023.

**Orthorectified Mosaic**: [Dropbox - Tumamoc Hill Feb 2023](https://www.dropbox.com/scl/fo/pem5nivwv9lgk6rgnc6c4/AAeaBOUbO4qgbshJguxBHZg?rlkey=zkfxodsst7x4ucsoq255pp0mw&e=1&dl=0)

**Source Images**: [Dropbox - Tumamoc Source Images](https://www.dropbox.com/scl/fo/cxb7mkl80f8ux9sfpfoty/AGhAKqDZ0MNjE4V0ZgrlI6M?rlkey=fu5b5hbr6jo6nn7zsknky4w60&e=1&st=blcns2qb&dl=0)

**Type**: Orthorectified mosaic  
**Date**: February 2023  
**Location**: Tumamoc Hill, Tucson, AZ
